Q：商户需满足什么条件？
A：
  1. 1、经工商行政管理机关（或主管机关）核准登记，依法经营，有良好的商业信誉；
  2. 2、在当地金融机构开立银行结算账户；
  3. 3、未被通报列入禁止发展商户名单；
  4. 4、满足上述条件的个体经营者及企业，均可成为iPayLinks支付服务的商户。
Q：成为商户需要办理哪些手续？
A：
  1. 拨打商户热线：400 665 0719，留下您的联系方式和基本资料，iPayLinks支付客户经理将主动联系您，为您答疑解惑、提供全程服务。其中涉及的主要手续包括：
  2. 1、 资质认证：提供资质认证文件、并完成实名认证；
  3. 2、 合同签约：根据您选择的产品和服务，签订纸质合同；
  4. 3、 生成账号：由iPayLinks支付客户经理根据您提供的信息，为您注册有效账号、开通服务。

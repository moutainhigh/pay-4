交易及系统安全策略 — 资系统安全保障支付系统正常运行，交易安全保障支付交易安全、快捷

·安全网络体系和安全稳定的系统
iPayLinks支付平台使用硬件防火墙，确保支付网关免于病毒与黑客攻击，避免由于网络产生的新漏洞受到威胁，为支付交易的机密数据提供保护措施，保障平台安全稳定。
·VERISIGN签发安全证书，128为SSL链路加密
iPayLinks支付采用国际认证体系VERISIGN颁发的全球网站鉴别和加密的SSL服务器证书，对用户信息、账户信息和签名都采用128位的SSL加密。 使用iPayLinks支付能最大程度保障支付信息的机密、支付过程的完整、商户及客户的合法身份、操作等信息的安全。
·支付交易订单采用MD5摘要认证
为保证用户传送到iPayLinks支付的订单不被恶意篡改，iPayLinks为每笔订单提供支付接口提供基于MD5的摘要认证。用户将订单信息及后台管理员密码等进行MD5摘要，将这些摘要连同订单支付信息一起发送到iPayLinks支付，iPayLinks支付通过MD5认证来判别该订单是否被篡改，从而保证用户的利益。

Q：什么是实名认证
A： 实名认证是iPayLinks支付为用户提供的一种身份识别服务，通过第三方机构提供的身份校验服务，完成iPayLinks支付账户与真实身份的关联，让账户交易更为安全
Q：实名认证有什么好处
A： 通过实名认证后，iPayLinks支付用户方可开启包括提现、大额收付款在内、完整的账户服务。
Q：如何申请实名认证
A：
  1. 1、 找到实名认证入口；

    登录后，在“我的账户”首页，点击账号状态中的实名认证图标，可开始申请； 或：进入“我的账户”—“账户信息”，在用户信息中，有一栏“实名状态”，

    点击“认证”即可进入申请流程

  2. 2、 填写实名信息； 填写您的姓名、身份证件；
  3. 3、 校验实名信息

    系统将递交至第三方机构校验身份；如姓名与证件不匹配，需重新填写；

  4. 4、 实名完成

    校验通过后，实名认证通过，您可使用提现、大额收付款等服务。
